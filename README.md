# Stock Price Prediction using RNN, CNN, and Hybrid CNN-RNN Models

## Overview

This project implements multiple deep learning models for stock price prediction using historical time-series data.

The implemented models include:

- Simple Recurrent Neural Network (RNN)
- 1D Convolutional Neural Network (CNN)
- Hybrid CNN + RNN Model

Apple (AAPL) and IBM stock datasets were used for training and evaluation.

## Features

- Time-series preprocessing
- Data visualization
- Sequence generation
- RNN implementation
- CNN implementation
- Hybrid CNN-RNN architecture
- Model comparison
- RMSE Evaluation

## Technologies

- Python
- TensorFlow
- Keras
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Results

The Hybrid CNN-RNN model demonstrated improved forecasting performance compared to standalone RNN and CNN models.
